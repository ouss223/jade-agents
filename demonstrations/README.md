# Demonstrations of the JADE Multi-Agent System

This folder contains complete demonstrations of the distributed monitoring system based on JADE, including detailed guides and multimedia resources from the actual deployment.

## Demonstration Structure

- **02_setup**: Guide for installing and configuring GNS3 infrastructure
- **03_network**: Network architecture and complete topology
- **04_agents**: Deployment and launching of JADE agents
- **05_db_grafana**: PostgreSQL and Grafana configuration
- **06_tests**: Load test results and validations

## Multimedia Resources

All demonstrations reference files present in the `/media/` directory containing screenshots and videos from the actual deployment in GNS3 environment on VMware.
